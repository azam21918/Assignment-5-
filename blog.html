Question-1: What are the different ways to select an element in the DOM?

The ways to select the DOM:

  • getElementById
  • getElementsByClassName
  • getElementsByTagName
  • querySelectorAll
  • querySelector

Question-2: What is the difference between innerHTML, innerText, and textContent ?

  • innerHTML used for to get the text with HTML tag
  • innerText used for to get only the text content with styles
  • textContent used for to get only the text content without styles

Question-3: What is event delegation in the DOM?

Controlling the DOM element from it's parent.

Question-4: What is event bubbling in the DOM?

Capturing the target element from the DOM tree and bubble up to the DOM.

Question-5: How do you create, add, and remove elements using JavaScript?

  • To create element=document.createElement()
  • To add an element=document.classList.add()
  • To remove element=document.classList.remove()
